JIRA TAM Report Script (Version 1.0)
Description:
This is a script used to Automate calculation of last external comments of JIRA issues given in the config file.
Requirements:
1. JIRA API Token : https://id.atlassian.com/manage-profile/security/api-tokens (configure it in Config/config.ini file)
2. JIRA Quary (configure it in Config/config.ini file) 
Python version : 3.9
Jira Python API : https://jira.readthedocs.io/installation.html
